# laravel-rules

Пытаемся унифицировать работу с laravel, выбрав и зафиксировав лучшие практики для работы с ним. Это не должны быть жёсткие рамки, но каждый раз, когда разработчик решает применить нестандартное решение, он должен уметь объяснить, почему он выбрал такое решение.
